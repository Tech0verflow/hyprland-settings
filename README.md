## Running the settings app

### Clone the git repository
```shell
git clone https://github.com/Tech0verflow/hyprland-settings.git
cd ./hyprland-settings
```
### Install the dependencies
```shell
pip install -r requirements.txt
```
### Run main.py
```shell
python main.py
```
